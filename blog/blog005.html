Standardizing AlphaMole's Structure & Colorful Chemistry

Today marks a significant step in AlphaMole's evolution. We focused on organizing the project for scalability and clarity, and on making the chemistry shine visually.

Key Updates
  • 📁 Created a standardized folder structure for CSS, JS, and Data.
  • 📦 Moved all assets into their respective folders for better organization.
  • 🧬 Added a structured elements.json file containing atomic information and color coding for each element.
  • 🎨 Updated the main page to use the color scheme from elements.json, so each atomic element is now displayed in its unique color.

With these changes, AlphaMole is not only easier to maintain, but also more visually engaging. The new color scheme brings the periodic table to life, making the learning experience more interactive and fun.
